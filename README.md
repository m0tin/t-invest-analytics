# T-Invest Analytics

Аналитический отчёт по портфелю Т-Инвестиций в Excel с pivot tables и графиками.

## Возможности

- Загрузка данных со всех брокерских счетов и ИИС через T-Invest API
- Excel-отчёт с 6 листами аналитики:
  - **Сводка** — дашборд с диаграммами (портфель по типам активов, секторам, доходы по месяцам, топ P&L)
  - **По счетам** — сводка по каждому счёту (стоимость, P&L, дивиденды, купоны, комиссии)
  - **Портфель** — все позиции с тикерами, ценами, P&L и условным форматированием
  - **Операции** — полная история сделок за 5 лет
  - **Дивиденды и купоны** — отфильтрованные доходные операции
  - **Комиссии** — агрегированные по месяцам
- Все листы оформлены как Excel Tables — удобно строить собственные pivot tables
- Условное форматирование (зелёный/красный) для P&L
- Круговые и столбчатые диаграммы

## Установка

```bash
git clone https://github.com/m0tin/t-invest-analytics.git
cd t-invest-analytics
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Настройка

1. Получите read-only API-токен в [настройках Т-Инвестиций](https://www.tbank.ru/invest/settings/)
2. Создайте файл `.env`:

```bash
cp .env.example .env
# Вставьте свой токен в .env
```

## Запуск

```bash
source venv/bin/activate
python build_report.py
```

Файл `t-invest-analytics.xlsx` будет создан в текущей директории.

## Безопасность

- Используется **read-only** токен — скрипт не может совершать сделки
- Токен хранится в `.env` и не попадает в git
- Все данные обрабатываются локально

## Требования

- Python 3.10+
- API-токен Т-Инвестиций (read-only)
